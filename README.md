# Take a rest 網路商城
### demo: https://jacky6450.github.io/Vue_Eshop/ 
### 網頁功能
### 前台
* 主頁-熱銷商品&類別介紹
* 賣場-商品陳列-商品分類過濾功能
* 單一商品詳細資訊
* 加入購物車
* 結帳-確認購物明細
* 結帳-填入購買人資料&使用優惠券
* 結帳-確認訂單&付款
### 後台
* 新增商品
* 修改商品
* 刪除商品
* 新增/管理優惠券
* 訂單列表
